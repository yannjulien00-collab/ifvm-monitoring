# Backend IFVM

Backend PHP organise pour accueillir une API REST simple.

## Dossiers

- `public/` : point d'entree web.
- `src/Config/` : configuration application et base de donnees.
- `src/Controllers/` : controleurs HTTP.
- `src/Core/` : noyau technique commun.
- `src/Middleware/` : verification auth/roles.
- `src/Models/` : objets metier.
- `src/Repositories/` : acces aux donnees.
- `src/Routes/` : declaration des endpoints.
- `src/Services/` : logique metier.
- `src/Utils/` : helpers techniques.
- `storage/` : logs et uploads.
- `tests/` : tests backend.
