# Frontend IFVM

Frontend React reserve au dashboard, a la carte interactive et aux formulaires metier.

## Dossiers

- `public/` : assets statiques.
- `src/assets/` : images, icones et ressources UI.
- `src/components/` : composants reutilisables.
- `src/features/` : modules fonctionnels par domaine.
- `src/layouts/` : shells et structures de pages.
- `src/pages/` : vues principales.
- `src/routes/` : configuration de navigation.
- `src/services/` : appels API.
- `src/styles/` : styles globaux.
- `src/utils/` : fonctions utilitaires.
